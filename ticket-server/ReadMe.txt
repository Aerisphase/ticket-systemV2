Эта серверная часть предоставляет REST API для:

Управления событиями (создание, чтение, обновление, удаление)

Бронирования билетов

Оплаты брони

Отмены бронирования

API доступно по адресу http://localhost:8080/api с двумя основными endpoint:

/events - для работы с событиями

/tickets - для работы с билетами
